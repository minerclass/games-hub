# Learning Games Hub

Public hub for Micah Miner's interactive educational games and learning labs.

The hub collects playable experiences connected to:

- Historical inquiry
- Source literacy
- Media ecology
- Pedagogical friction
- Generative AI and learning design

## Live Site

After publication, the hub is available at:

`https://minerclass.github.io/games-hub/`

## Included Games

- [Keepers of Inquiry](https://minerclass.github.io/historical-inquiry-friction/keepers-of-inquiry/)
- [Keeper of the Source](https://minerclass.github.io/source_game/)
- [Friction Lab](https://minerclass.github.io/friction-game/)
- [Keeper of the Word](https://minerclass.github.io/orality_game/)

## Run Locally

Open `index.html` in a browser or serve the folder with a static server:

```bash
python -m http.server 8000
```

## Privacy

This repository contains public links and static hub content only. It does not collect, transmit, or store participant, student, staff, or research-subject data.
